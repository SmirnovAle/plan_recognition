# Распознавание плана квартиры из изображения

Прототип для извлечения архитектурной геометрии из изображений планов квартир.

## Функциональность
- Детекция стен на планах квартир
- Экспорт результатов в JSON формат
- Визуализация найденных стен

## Требования
- Python 3.8+
- Зависимости: см. `requirements.txt`

## Установка
```bash
git clone <repository-url>
cd plan_recognition
pip install -r requirements.txt
